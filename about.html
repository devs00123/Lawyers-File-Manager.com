About

This Lawyer Case Manager stores case records and attachments locally using IndexedDB. Key features:

  • Add / edit / delete cases
  • Attach multiple files (images, PDFs) and preview/download them
  • Auto Next Hearing Date (Current + 30 days)
  • Auto-advance when current hearing date passes (moves current & calc next)
  • Powerful search, sort & filtering
  • Export / Import JSON for backup

For cloud sync or notifications, I can add a server API next.
